# QR コードジェネレーター

iPhone・Androidの自由記述文章、URL、Wifi設定、連絡先（vCard形式）のQRコードを生成します。
「URLコピー」ボタンで現在入力している内容を指定するURLパラメター付きURLをクリップボードにコピーできます。
